Service Worker Launch Demo

This is a demo site for the Service Worker "launch" event polyfill.

Because "launch" is not implemented in the browser, you need to use the Navigation Simulator. Think of that as a test UI for generating incoming navigations into your website.

This page acts as the "main page" of the site; it uses the "launch" event to steal any navigations to the "/compose" URL.

Note: The demo is unable to focus an existing tab or open a new one (due to user gesture restriction). Look for error messages in the console to imagine what is "supposed" to happen (or patch Chromium with this hack to remove the gesture check).
